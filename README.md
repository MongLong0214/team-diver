## 시작하기

Git 저장소를 Pull 한 다음...

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
로컬 실행 도메인 : http://localhost:3000 

메인 페이지 주소 : http://localhost:3000/main

## 앱 실행 
"생성하기"를 누르면 브라우저 콘솔에 선택된 프롬포트와 이미지 base64 데이터가 출력됩니다. 

## 비고
진행사항 보고 : 
- 스타일은 아직 확정되지 않아, 기본 HTML 요소들로 구성했습니다.
- 이미지는 S3 로 업로드 하지 않으며 이미지 업로드는 본인 기기에서 선택합니다.
